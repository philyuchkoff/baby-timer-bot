# 🤰 baby-timer-bot

Telegram-бот для расчёта акушерского срока, предполагаемой даты родов (ПДР) и даты выхода в декрет по российским медицинским и юридическим стандартам.

## 🚀 Возможности

- 📅 Расчёт ПДР по формуле Негеле (первый день последних месячных + 280 дней)
- 🤰 Определение текущего акушерского срока (недели + дни)
- 👔 Расчёт даты начала декретного отпуска (30 недель - одноплодная, 28 недель - многоплодная)
- 💰 Калькулятор декретных выплат с проверкой на мин./макс. лимиты 2026 года

## 🛠️ Технологии

- Go 1.21+
- gopkg.in/telebot.v3
- Формула Негеле
- МРОТ 2026: 27 093 ₽

## 🏃‍♂️ Быстрый старт

```bash
git clone https://github.com/ваш_никнейм/baby-timer-bot.git
cd baby-timer-bot
cp .env.example .env  # добавьте TELEGRAM_BOT_TOKEN
go mod download
go run cmd/bot/main.go
```

## 📝 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/pdr` | Рассчитать ПДР и срок беременности |
| `/benefit` | Рассчитать декретные выплаты |

## 📚 Источники

- Приказ Минздрава РФ № 1130н
- Формула Негеле (стандарт акушерства)
- Постановление Правительства РФ № 1234 (лимиты 2026)
